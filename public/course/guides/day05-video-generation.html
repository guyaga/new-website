Day 05

יצירת וידאו עם AI

fal.ai MCP

בשיעור הזה נלמד ליצור סרטוני AI מקצועיים ישירות מ-Claude Code — מטקסט או מתמונה, עם המודלים הטובים בעולם: Veo 3.1, Kling 3, Seedance ועוד. נכיר גם מה זה MCP ואיך הוא הופך את Claude לכלי חזק יותר.

מה זה MCP?

תוספים ל-Claude Code

MCP (Model Context Protocol) זה כמו תוסף (Plugin) ל-Claude Code. בדיוק כמו שתוספים לכרום מוסיפים יכולות לדפדפן, MCP מוסיף יכולות ל-Claude. במקרה הזה — גישה למאות מודלים של יצירת וידאו דרך fal.ai.

פקודה אחת להתקנה. ואז Claude יכול ליצור סרטונים.

כמה זה עולה?

תמחור

חשוב לדעת

יצירת וידאו עם AI עולה כסף לכל שנייה. סרטון של 5 שניות יכול לעלות $0.35 עד $2. זה עדיין זול דרמטית בהשוואה לצילום מסורתי (צלם, שחקנים, עריכה), אבל חשוב להיות מודעים לעלויות.

מודלמחיר/שנייהלמה מתאים
Veo 3.1$0.20-0.40האיכות הכי גבוהה, המודל של גוגל
Kling 3 Pro$0.11-0.20קולנועי, עם אודיו מובנה
Kling 2.5 Turbo~$0.07יחס מצוין של מהירות/איכות
Veo 3.1 Liteזול יותרגרסה חסכונית של Veo
טיפ חיסכון

התחילו עם מודלים זולים (Kling 2.5, Veo Lite) לבדיקות. רנדרו סופי עם מודל פרימיום רק כשהפרומפט והתמונה מושלמים.

הגדרות

איך להתחיל

1

צרו חשבון ב-fal.ai

לכו ל-fal.ai והירשמו.

2

צרו API Key

לכו ל-fal.ai/dashboard/keys ולחצו Create Key.

3

הוסיפו קרדיטים

לכו ל-fal.ai/dashboard/usage-billing/credits והוסיפו $5-10 להתחלה.

4

התקנת fal MCP (בטרמינל — לא בתוך Claude Code!)

סגרו את Claude Code קודם. פתחו טרמינל רגיל (PowerShell, Terminal) והריצו:

Run in Terminal (NOT inside Claude Code)
claude mcp add --transport http fal-ai https://mcp.fal.ai/mcp --header "Authorization: Bearer YOUR_FAL_KEY"

החליפו YOUR_FAL_KEY במפתח האמיתי שלכם משלב 2.

למה בטרמינל ולא ב-Claude Code?

פקודת claude mcp add מגדירה את Claude Code עצמו — הוא לא יכול להגדיר את עצמו מבפנים. כמו התקנת תוסף לדפדפן — מתקינים מבחוץ.

5

הפעילו מחדש + התקנת סקיל

סגרו ופתחו מחדש את Claude Code (ה-MCP נטען בהפעלה). אחר כך הדביקו:

Paste into Claude Code
Install the fal-ai-video-generation skill from https://github.com/guyaga/10d10s-day04-video-generation and verify the fal MCP is connected by searching for available video models.
מה אפשר לעשות

4 תהליכי עבודה

WORKFLOW 01

טקסט → וידאו

תארו מה אתם רוצים לראות — ה-AI יוצר את הסרטון.

Generate a 5-second video using Veo 3.1: "A slow aerial drone shot over a futuristic city at sunset, neon lights reflecting on glass buildings, cinematic"
WORKFLOW 02

תמונה → וידאו (התוצאות הכי טובות)

צרו תמונה עם סקיל יום 1, ואז הנפישו אותה לסרטון.

1. First, use nano-banano-pro to generate a still frame: "Professional woman presenting at tech conference, dramatic stage lighting" 2. Then animate it with Kling 3: "She gestures confidently, camera slowly zooms in, subtle audience movement"
WORKFLOW 03

פריים ראשון + אחרון (שליטה מלאה)

צרו תמונת פתיחה וסיום — ה-AI ייצור את התנועה ביניהן.

1. Generate first frame: "A closed laptop on a desk, morning light" 2. Generate last frame: "Same laptop now open with dashboard, coffee cup appeared" 3. Use Veo 3.1 first-last-frame-to-video to generate the transition
WORKFLOW 04

צינור הפקה מלא (ימים 1-5)

השתמשו בכל מה שלמדתם עד עכשיו ליצירת סרטון מלא.

Let's create a complete product video: 1. Generate a hero image (Day 1 - nano-banano-pro) 2. Animate it into 5-second video (Day 5 - Kling 3) 3. Check quality (Day 2 - ai-video-analyzer) 4. Add subtitles and resize (Day 3 - ai-video-editor) 5. Add voiceover (Day 4 - ai-voice-audio)
לפני שמייצרים

צ'קליסט — כדי לא לבזבז כסף

עברו על הרשימה הזו לפני כל יצירת וידאו:

  • הפרומפט ספציפי? — תנועה, מצלמה, מצב רוח, אורך
  • התמונה באיכות גבוהה? — מינימום 2K לתמונה-לוידאו
  • המודל מתאים? — זול לבדיקות, פרימיום לסופי
  • האורך סביר? — התחילו עם 4-5 שניות לבדיקה
  • יש מספיק קרדיטים? — בדקו ב-fal.ai dashboard
עצה חשובה

תהיו מעורבים בכל שלב — בדקו את איכות התמונה, את הפרומפט, את המודל שנבחר. כל דבר שמשפיע על התוצאה שווה בדיקה לפני היצירה. אל תסמכו על Claude באופן עיוור — זה הכסף שלכם.

טיפים לפרומפטים

איך לכתוב פרומפט טוב לוידאו

1

תארו תנועה

"המצלמה נעה לאט שמאלה, רוח מנשבת בעצים, ציפורים עוברות ברקע"

2

ציינו מצלמה

3

הגדירו אורך ומצב רוח

"8 שניות, סינמטי, תאורה דרמטית" או "5 שניות, אנרגטי, סושיאל מדיה"

4

לתמונה-לוידאו: תארו מה משתנה

אל תחזרו על מה שכבר נראה בתמונה. תארו רק את התנועה החדשה.

Day 08

הודעות וואטסאפ

Green API

בשיעור הזה נלמד לשלוח ולקבל הודעות וואטסאפ ישירות מ-Claude Code — טקסט, קבצים, תמונות, מיקום ועוד. נבין מה זה Green API, מה הסיכונים, ואיך להשתמש בזה בצורה בטוחה.

חשוב לקרוא קודם

מה זה Green API?

Green API הוא שירות שמחבר את Claude Code לוואטסאפ שלכם — בדיוק כמו WhatsApp Web. זה לא ה-API הרשמי של Meta.

לא רשמי (Unofficial)
Green API עובד דרך "מכשיר מקושר" — כמו WhatsApp Web. זה לא ה-API הרשמי של מטא. יותר פשוט וזול, אבל צריך להיזהר.
הטלפון חייב להיות דלוק
הטלפון שלכם חייב להיות מחובר לאינטרנט כל הזמן. אם הטלפון נכבה — החיבור נפסק.
סיכון חסימה
אם תשתמשו בצורה לא זהירה (הודעות ספאם, שליחה מהירה) — וואטסאפ יכול לחסום את המספר שלכם.
בשימוש זהיר — מדהים
שליחת תזכורות, שיתוף קבצים, עדכון אנשי קשר מוכרים — כל זה עובד מעולה ובטוח.
השורה התחתונה

בשימוש זהיר, Green API הוא כלי עוצמתי ומועיל מאוד. בשימוש רשלני, אפשר לאבד את חשבון הוואטסאפ. הסקיל הזה בנוי עם הגנות בטיחות — עקבו אחריהן.

בטיחות

מה בטוח ומה מסוכן

✅ בטוח
הודעות לעצמכם, קריאת הודעות נכנסות, בדיקת היסטוריית צ'אט, בדיקת סטטוס, אנשי קשר מוכרים
🚫 מסוכן
שליחת הודעות זהות לאנשים רבים, שליחה מהירה בלי השהייה, יצירת קבוצות, שליחה למספרים לא מוכרים

כללי זהב

  • תמיד אשרו לפני שליחה — Claude יראה לכם את ההודעה קודם
  • לא לשלוח אותה הודעה לאנשים שונים — תגוונו
  • מקסימום 200 הודעות ביום, מינימום 15 שניות בין הודעות
  • חממו מספר חדש — 10 ימים של שימוש הדרגתי
  • אם הודעות מאטות — עצרו מיד ל-24-48 שעות

הגדרות

איך להתחיל

1

צרו חשבון

לכו ל-green-api.com והירשמו.

2

צרו Instance

ב-Console, צרו Instance חדש. תקבלו API URL, Instance ID ו-Token.

3

חברו את הוואטסאפ

פתחו וואטסאפ → הגדרות → מכשירים מקושרים → קישור מכשיר. סרקו את ה-QR ב-Console.

4

התקנת הסקיל

Paste into Claude Code
Install the whatsapp-messenger skill from https://github.com/guyaga/10d10s-day08-whatsapp and help me set up my Green API credentials (GREEN_API_URL, GREEN_API_INSTANCE, GREEN_API_TOKEN) as environment variables.
5

הגדירו רשימת אנשי קשר (מומלץ)

שמרו את אנשי הקשר כדי לשלוח לפי שם:

contacts.json
[ {"name": "אני", "phone": "972541234567"}, {"name": "אמא", "phone": "972549876543"} ]
חימום מספר

מספר חדש? חובה לחמם

אם אתם מחברים מספר חדש או כזה שלא היה פעיל — חובה לעבור תהליך חימום של 10 ימים.

1

יום 1

הירשמו, הגדירו תמונת פרופיל ושם. חכו 24 שעות.

2

ימים 2-4

רק קבלו הודעות מחברים (הודעה כל שעתיים). לא לשלוח.

3

ימים 4-7

התחילו לענות להודעות נכנסות. 12-20 הודעות ביום, רק לאנשי קשר מוכרים.

4

ימים 8-30

הגדילו בהדרגה עד 200 הודעות ביום. גוונו תוכן (טקסט, תמונות, קול).

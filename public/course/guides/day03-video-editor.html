Day 03

עריכת וידאו עם AI

ffmpeg + yt-dlp

בשיעור הזה נלמד לערוך וידאו ישירות מ-Claude Code — חיתוך, מיזוג, הסרת שקטים, יצירת Reels, הוספת כתוביות, הורדה מיוטיוב, חילוץ אודיו, ועריכה אוטומטית של פודקאסטים. בלי תוכנת עריכה, בלי Timeline.

מה זה בכלל?

ffmpeg + yt-dlp

ffmpeg הוא הכלי הכי חזק בעולם לעריכת וידאו דרך שורת פקודה. כל תוכנת עריכה משתמשת בו מאחורי הקלעים. yt-dlp הוא כלי להורדת סרטונים מיוטיוב ומאות אתרים נוספים.

היופי? אתם לא צריכים לדעת את הפקודות. אתם מתארים מה אתם רוצים ב-Claude Code — והוא כותב ומריץ את הפקודות בשבילכם.

חיתוך מדויק
חתכו כל קטע בדיוק של שנייה. "מ-0:30 עד 1:45" — ומקבלים קליפ מושלם.
הסרת שקטים
זיהוי אוטומטי של רגעי שקט והסרתם. מושלם לפודקאסטים ולשיחות מוקלטות.
יצירת Reels/Shorts
חיתוך, שינוי גודל ל-9:16, והוספת כתוביות — הכל בפקודה אחת.
הורדה מיוטיוב
הורידו כל סרטון ציבורי — או רק את האודיו כ-MP3.
מיזוג קליפים
חברו מספר קבצי וידאו לסרטון אחד עם מעברים חלקים.
עיבוד קבוצתי
ערכו 100 קבצים בבת אחת — שנו גודל, פורמט, או הוסיפו כתוביות לכולם.
בלי עלות

ffmpeg ו-yt-dlp הם כלים חינמיים לגמרי שרצים על המחשב שלכם. אין צורך ב-API Key, אין טוקנים, אין חיוב.

הגדרות

התקנה

1

התקנת ffmpeg

Windows: winget install FFmpeg
macOS: brew install ffmpeg

2

התקנת yt-dlp

כל הפלטפורמות: pip install yt-dlp

3

התקנת הסקיל

פתחו Claude Code והדביקו:

Paste into Claude Code
Install the ai-video-editor skill from https://github.com/guyaga/10d10s-day03-video-editor and verify that ffmpeg and yt-dlp are available on my system

מה אפשר לעשות

7 דברים שתוכלו לעשות

USE CASE 01

חיתוך סרטון

חתכו בדיוק את הקטע שאתם צריכים — בלי לפתוח תוכנת עריכה.

Trim my video at D:/Videos/interview.mp4 — keep only from 2:30 to 8:15
USE CASE 02

הסרת שקטים מפודקאסט

הסירו את כל השקטים הארוכים וקבלו פודקאסט חד וזורם.

Remove all silences longer than 1 second from D:/Podcast/episode.mp4 and normalize the audio
USE CASE 03

יצירת Reel מסרטון ארוך

קחו סרטון של 30 דקות וצרו ממנו Reel של 30 שניות — אנכי עם כתוביות.

Create a 30-second vertical reel from D:/Videos/webinar.mp4 starting at 15:00. Crop to 9:16 and add subtitles.
USE CASE 04

הורדת MP3 מיוטיוב

הורידו רק את האודיו מכל סרטון ציבורי ביוטיוב.

Download the audio from https://youtube.com/watch?v=xxxxx as a high quality MP3
USE CASE 05

מיזוג קליפים

חברו מספר קבצים לסרטון אחד עם מעברים חלקים.

Merge all MP4 files in D:/Project/clips/ into one video with 1-second fade transitions between each
USE CASE 06

הוספת כתוביות

תמללו את הסרטון וצרפו כתוביות ישירות לתוך הוידאו.

Transcribe D:/Videos/talk.mp4 and burn the subtitles into the video with big white text, Instagram style
USE CASE 07

עיבוד קבוצתי

ערכו תיקייה שלמה של קבצים בבת אחת.

Resize all videos in D:/Clips/ to 1080x1920 vertical format and save to D:/Clips/vertical/

טיפים

איך לקבל תוצאות טובות

1

תנו נתיב מלא לקובץ

"D:/Videos/my-file.mp4" — לא "my-file.mp4". ככה Claude מוצא את הקובץ.

2

ציינו חותמות זמן מדויקות

"מ-2:30 עד 5:15" עדיף על "את האמצע". פורמט: MM:SS או HH:MM:SS.

3

השתמשו בסקיל ניתוח וידאו קודם

אם יש לכם את סקיל ניתוח הוידאו (יום 2), נתחו את הסרטון לפני העריכה כדי למצוא את הרגעים הטובים.

4

בדקו על קליפ קצר קודם

לפני עיבוד של סרטון של שעה — בדקו על קטע של 30 שניות שהתוצאה טובה.

פתרון בעיות

משהו לא עובד?

?

"ffmpeg not found"

התקינו ffmpeg וודאו שהוא ב-PATH. ב-Windows: winget install FFmpeg

?

הקובץ ריק (0 bytes)

בדקו שחותמות הזמן בטווח של הסרטון. אם הסרטון 5 דקות, אי אפשר לחתוך ב-10:00.

?

אין אודיו אחרי העריכה

הוסיפו -c:a copy או -c:a aac לפקודה.

?

הורדה מיוטיוב נכשלת

עדכנו yt-dlp: pip install --upgrade yt-dlp

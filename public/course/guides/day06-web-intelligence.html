Day 06

אינטליגנציית רשת

Firecrawl

בשיעור הזה נלמד לגרד (לסרוק) כל אתר ולהוציא ממנו מידע מובנה — פרופיל מותג מלא, צבעים, לוגו, מסרים שיווקיים, ניתוח מתחרים ומחקר תוכן. הכל ישירות מ-Claude Code.

מה זה בכלל?

הכירו את Firecrawl

Firecrawl הוא שירות שיודע לקרוא כל אתר אינטרנט ולהפוך אותו לטקסט נקי ומובנה. במקום לגלוש, להעתיק ולהדביק — אתם נותנים URL ומקבלים את כל המידע מאורגן.

טקסט נקי
מקבלים Markdown נקי — בלי HTML, בלי פרסומות, בלי רעש. רק התוכן.
חילוץ מובנה
מבקשים שדות ספציפיים (שם, צבעים, לוגו) ומקבלים JSON מסודר.
סריקת אתר שלם
לא רק דף אחד — סורקים 5-10 דפים ומקבלים תמונה מלאה של המותג.
הורדת נכסים
מוצאים לוגו, תמונות, favicon — ושומרים אותם מקומית.
ניתוח מתחרים
סורקים 3 מתחרים ויוצרים טבלת השוואה אוטומטית.
500 דפים חינם
ה-Free Tier נותן 500 דפים בחודש. מספיק ל-50-100 פרופילי מותג.
כמה זה עולה?

תמחור

ב-Free Tier אפשר לבנות 50-100 פרופילי מותג בחודש. מספיק בהחלט ללמידה.

הגדרות

איך להתחיל

1

צרו חשבון

לכו ל-firecrawl.dev והירשמו.

2

צרו API Key

ב-Dashboard, העתיקו את ה-API Key.

3

הגדירו את המפתח

Windows (PowerShell)
$env:FIRECRAWL_API_KEY="your-key-here"
macOS / Linux
export FIRECRAWL_API_KEY=your-key-here
4

התקנת הסקיל

פתחו Claude Code והדביקו:

Paste into Claude Code
Install the web-intelligence skill from https://github.com/guyaga/10d10s-day06-web-intelligence and install firecrawl-py with pip. Help me set up my FIRECRAWL_API_KEY.
מה אפשר לעשות

4 דברים שתוכלו לעשות

USE CASE 01

בניית פרופיל מותג

תנו URL של אתר — קבלו פרופיל מלא: שם, צבעים, לוגו, טון דיבור, קהל יעד, מסרים שיווקיים.

Build a complete brand profile from https://stripe.com — extract name, tagline, colors (hex), fonts, logo, target audience, brand voice. Download the logo. Save to ~/brand-profiles/stripe/
USE CASE 02

ניתוח מתחרים

השוו בין 3 מתחרים — מה הם מציעים, למי, באיזה מחיר.

Compare these 3 competitors: - https://notion.so - https://linear.app - https://height.app Extract: value proposition, key features, pricing, target audience. Output as a comparison table.
USE CASE 03

מחקר תוכן

סרקו בלוג של מתחרה — הבינו את אסטרטגיית התוכן שלהם.

Scrape the blog at https://example.com/blog — extract all post titles, dates, topics. What's their publishing frequency and content strategy?
USE CASE 04

סריקת דף כ-Markdown

קבלו את תוכן הדף כטקסט נקי — מושלם למחקר, סיכום, או שימוש חוזר.

Scrape https://example.com/about as clean markdown. I need the content for a research document.
מגבלות

מה Firecrawl לא יכול

  • תוכן סושיאל מדיה — אינסטגרם, טיקטוק, לינקדאין חסומים. לזה יש Apify MCP (אופציונלי, שירות נפרד)
  • דפים שדורשים התחברות — תוכן מאחורי Login לא נגיש
  • אתרים שחוסמים סורקים — אתרים גדולים עם הגנת anti-bot
טיפ בונוס

אם אתם צריכים תוכן מרשתות חברתיות, חפשו את Apify MCP — שירות שמתמחה בגרידת סושיאל מדיה. זה לא חלק מהשיעור הזה, אבל טוב לדעת שזה קיים.

פתרון בעיות

משהו לא עובד?

?

"API key not found"

ודאו שהגדרתם FIRECRAWL_API_KEY ב-Terminal.

?

"firecrawl-py not installed"

הריצו: pip install firecrawl-py

?

הדף חוזר ריק

האתר עשוי לחסום סורקים. נסו דף אחר או אתר אחר.

?

הלוגו לא נמצא

בדקו את האתר ידנית — הלוגו עשוי להיות SVG inline או CSS background.

תמחור Upload Post

השירות שמאפשר לפרסם מ-Claude Code ישירות לכל הפלטפורמות.

חבילהמחירמה כלול
5 ערוצים~$255 מותגים — כל מותג מחובר לכל 11 הפלטפורמות
מה זה "ערוץ"?

מותג = כל 11 הפלטפורמות מחוברות. לדוגמה: "העסק שלי" = מותג 1 עם אינסטגרם + לינקדאין + פייסבוק + טיקטוק + X + Threads + כו'. עם 5 מותגים מכסים כמה עסקים או לקוחות.

הגדרות

איך להתחיל

1

הירשמו ל-Upload Post

לכו ל-upload-post.com והירשמו.

2

חברו את הערוצים

ב-Dashboard, חברו את חשבונות הסושיאל שלכם (אינסטגרם, לינקדאין, וכו').

3

קבלו API Key

ב-Dashboard, העתיקו את ה-API Key.

4

הגדירו את המפתח

Terminal
export UPLOAD_POST_API_KEY=your-key-here
5

התקנת הסקיל

Paste into Claude Code
Install the social-media-publisher skill from https://github.com/guyaga/10d10s-day07-social-media and help me set up my UPLOAD_POST_API_KEY for publishing to social media.
מה אפשר לעשות

4 תהליכי עבודה

WORKFLOW 01

עיצוב פוסט ממותג

צרו פוסט מעוצב עם הצבעים והלוגו שלכם — מוכן לפרסום.

Create a 1:1 Instagram post for my brand announcing "New Product Launch". Use my brand colors #111111, #E63B2E, #F5F3EE. Include my logo. Save to ~/social-posts/launch.png
WORKFLOW 02

יצירת קרוסלה

צרו סדרת שקפים (7 שקפים) לאינסטגרם או לינקדאין — עם Hook, תוכן, ו-CTA.

Create a 7-slide Instagram carousel about "5 Ways AI Saves 10 Hours/Week". Hook slide, 5 tips (1 per slide), CTA slide. My brand colors. 1080x1350 size.
WORKFLOW 03

פרסום למספר פלטפורמות

פרסמו את אותו תוכן לאינסטגרם, לינקדאין וטוויטר בו-זמנית.

Post ~/social-posts/launch.png to Instagram, LinkedIn, and Twitter with caption: "We just launched something big. Link in bio."
WORKFLOW 04

תזמון פוסט

קבעו פרסום למחר בשעה 10 בבוקר — לא צריך לזכור לפרסם.

Schedule ~/social-posts/carousel/ as an Instagram carousel for tomorrow at 10:00 AM Israel time with caption: "5 tips that changed my workflow 👇"

קרוסלות מקבלות פי 2-3 יותר engagement מפוסט בודד באינסטגרם ולינקדאין. תמיד העדיפו קרוסלה על תמונה בודדת.

כללים קריטיים

לפני שמפרסמים

  • תמיד תבדקו את התמונה לפני הפרסום — Claude יראה לכם
  • תמיד תאשרו את הכיתוב — קראו את הטקסט לפני שליחה
  • לעולם אל תפרסמו בלי אישור מפורש — זה תוכן ציבורי אמיתי
  • בדקו לאיזה חשבון הולך הפוסט (אם יש לכם כמה)
  • התאימו את הגודל לפלטפורמה (1:1 לאינסטגרם, 16:9 לטוויטר)

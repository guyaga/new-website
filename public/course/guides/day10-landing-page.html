Day 10 — The Finale

בניית אתר נחיתה ופרסום

React + Vite + Tailwind + GSAP + Netlify

השיעור האחרון והכי מרגש. בנו אתר נחיתה מקצועי עם אנימציות קולנועיות, עיצוב ממותג, תמונות AI, אופטימיזציה ל-SEO ול-AI, ופרסום לאינטרנט בפקודה אחת. כל מה שלמדתם בימים 1-9 מתחבר כאן.

מילון מושגים

מושגים שכדאי להכיר

Node.js
תוכנה שמריצה JavaScript על המחשב שלכם (לא רק בדפדפן). הבסיס לכל הפרויקט.
npm
מנהל חבילות. מתקין קוד שאחרים כתבו. מגיע עם Node.js.
Vite
הכלי שבונה את האתר שלכם. הופך קוד React לקבצים שדפדפן מבין.
React
ספרייה לבניית ממשקי משתמש. הדף שלכם בנוי מ"רכיבים" של React.
Tailwind CSS
דרך לעצב את הדף עם שמות מחלקות במקום לכתוב CSS בנפרד.
GSAP
ספריית אנימציות. מה שגורם לדברים לזוז בצורה חלקה וקולנועית.
Netlify
שירות אחסון חינמי. שם את האתר שלכם באינטרנט עם כתובת URL.
Deploy
העלאת האתר לאינטרנט. אחרי Deploy כל אחד יכול לגלוש באתר.

B: Midnight Luxe
מועדון פרטי + אטלייה של שענים. שחור, זהב, שנהב. מתאים לפיננסים, יוקרה.
C: Brutalist Signal
חדר בקרה עתידני. ביז', אדום, שחור. מתאים לטכנולוגיה, AI, ייעוץ.
D: Vapor Clinic
מעבדת גנום בטוקיו. שחור עמוק, סגול, לבן. מתאים ל-SaaS, קריפטו, חדשנות.

פרסום

איך מפרסמים באינטרנט

אחרי ש-Claude בנה את האתר, שתי פקודות ואתם באוויר:

Terminal
npm run build netlify deploy --prod --dir=dist

בפעם הראשונה Netlify ישאל אם ליצור אתר חדש. תגידו כן ותבחרו שם.

אם יש לכם דומיין (כמו mybrand.com), הריצו netlify domains:add mybrand.com ועקבו אחרי ההוראות. Netlify מנחה אתכם בהגדרת ה-DNS.

SEO + AI

אופטימיזציה לגוגל ול-AI

Claude מוסיף את כל האופטימיזציות אוטומטית:

  • תגי Meta — כותרת, תיאור, מילות מפתח לגוגל
  • Open Graph — תצוגה מקדימה יפה כשמשתפים בסושיאל
  • JSON-LD — מידע מובנה שגוגל מבין (סוג עסק, שירותים, מיקום)
  • llms.txt — קובץ טקסט שמסביר ל-AI מה האתר עושה (ChatGPT, Perplexity, Claude)
  • HTML סמנטי — תגי header, main, section, article, nav — מבנה ש-AI ומנועי חיפוש מבינים
  • נגישות — alt text על תמונות, aria labels, skip-to-content

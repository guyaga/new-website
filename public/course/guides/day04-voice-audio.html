Day 04

קול ואודיו עם AI

ElevenLabs

בשיעור הזה נלמד ליצור קריינות מקצועית, מוזיקה ואפקטים קוליים ישירות מ-Claude Code. נשכפל את הקול שלנו, ניצור נרטיב לסרטונים, ונוסיף שכבת אודיו מקצועית לכל פרויקט.

מה זה בכלל?

הכירו את ElevenLabs

ElevenLabs הוא שירות AI שמייצר קול אנושי טבעי מטקסט. המודל V3 החדש שלהם כמעט בלתי ניתן להבחנה מדיבור אמיתי. אפשר גם לשכפל את הקול שלכם — ואז ה-AI ידבר בדיוק כמוכם.

מודל V3 — הכי טבעי
הקול הכי טבעי שנוצר אי פעם על ידי AI. כמעט בלתי אפשרי להבדיל מדיבור אמיתי.
שכפול קול
הקליטו דקה של הקול שלכם — ה-AI ידבר כל מה שתכתבו בקול שלכם בדיוק.
32+ שפות
אותו קול, שפה אחרת. עברית, אנגלית, ספרדית — הכל עובד.
יצירת מוזיקה
צרו פסקול מותאם אישית — לו-פי, סינמטי, אלקטרוני, כל סגנון.
אפקטים קוליים
צרו צלילי התראה, מעברים, אפקטי סביבה — בתיאור טקסט.
שליטה ברגש
כוונו את הקול — דרמטי, רגוע, אנרגטי, מקצועי. הכל בהגדרות פשוטות.
כמה זה עולה?

תמחור

שירות זה דורש חשבון ב-ElevenLabs עם API Key נפרד (לא אותו מפתח מימים 1-2).

תוכניתמחירמה כלול
Freeחינם~10,000 תווים/חודש (5-7 דקות דיבור)
Starter$5/חודש30,000 תווים + 3 שכפולי קול
Creator$22/חודש100,000 תווים + 10 שכפולים
ה-Free מספיק

10,000 תווים זה מספיק ל-5-7 דקות של דיבור. בהחלט מספיק ללמידה ולתרגול השיעור הזה.

הגדרות

איך להתחיל

1

צרו חשבון

לכו ל-elevenlabs.io ולחצו Sign Up.

2

צרו API Key

לחצו על אייקון הפרופיל (למטה-שמאל) → Profile + API key → Create API Key.

3

הגדירו את המפתח

פתחו Terminal:

Windows (PowerShell)
$env:ELEVEN_API_KEY="your-api-key-here"
macOS / Linux
export ELEVEN_API_KEY=your-api-key-here
4

התקנת הסקיל

פתחו Claude Code והדביקו:

Paste into Claude Code
Install the ai-voice-audio skill from https://github.com/guyaga/10d10s-day04-voice-audio and help me set up my ElevenLabs API key
מה אפשר לעשות

5 דברים שתוכלו ליצור

USE CASE 01

קריינות לסרטון

צרו קריינות מקצועית לכל סרטון — הדרכה, פרסומת, קורס.

Generate a voiceover saying "Welcome to our AI marketing platform. In this video, we'll show you how to transform your content strategy." Use a warm professional voice. Save to D:/Audio/narration.mp3
USE CASE 02

שכפול הקול שלכם

הקליטו דקה של הקול שלכם — ותנו ל-AI לדבר כל מה שתכתבו בקול שלכם.

Clone my voice from D:/Audio/my-voice-sample.mp3 then use it to narrate: "Welcome to this week's episode. I'm Guy, and today we're talking about AI tools."
USE CASE 03

מוזיקת רקע

צרו פסקול מותאם אישית לכל פרויקט.

Generate 30 seconds of calm lo-fi hip hop music with soft piano and light drums, perfect for a YouTube video background
USE CASE 04

אפקטים קוליים

צרו צלילים מותאמים לאפליקציה, סרטון או פרזנטציה.

Generate a modern, clean notification chime sound for a tech app, 2 seconds long
USE CASE 05

קריינות + וידאו (יום 3 + יום 4)

צרו קריינות ושלבו אותה ישירות לתוך סרטון.

Generate a voiceover for this script, then add it to D:/Videos/demo.mp4 as narration: "Introducing the future of content creation."
שכפול קול

טיפים לתוצאות הכי טובות

1

הקליטו 1-3 דקות

מינימום 30 שניות, אבל ככל שתקליטו יותר — התוצאה טובה יותר.

2

סביבה שקטה

כבו מזגן, מאוורר, טלוויזיה. רעשי רקע הורסים את השכפול.

3

דברו טבעי

אל תקראו כמו רובוט. דברו כמו שאתם מדברים בשגרה.

4

מיקרופון טוב

טלפון בחדר שקט עובד. מיקרופון USB עדיף.

טיפ מקצועי

כשמשתמשים בקול משוכפל, העלו את הגדרת ה-similarity_boost ל-0.85 — ככה זה יישמע יותר כמוכם.

Day 09

מצגות AI

python-pptx + Gemini 3 Pro

בשיעור הזה נלמד ליצור מצגות PowerPoint ברמה מקצועית ישירות מ-Claude Code — עם מבנה סיפורי, תמונות שנוצרו ב-AI, עיצוב ממותג, ו-19 סוגי שקפים שונים. תנו נושא — ותקבלו מצגת של 20+ שקפים.

שתי דרכים

איך זה עובד

מהיר — "תנו לי נושא"
"צרו מצגת של 20 שקפים על AI בשיווק" — Claude בוחר מסגרת סיפורית, מייצר תמונות, ובונה את כל השקפים.
מותאם — "אני רוצה שליטה"
הגדירו מותג, בחרו מסגרת סיפורית, ספקו קבצים עם נתונים, וכוונו כל שלב.
חומר מקור

ספקו תוכן לתוצאות טובות יותר

Claude מצוין במבנה ועיצוב, אבל הידע שלכם הופך את התוכן למדויק.

מצגת קיימת
Claude יקרא וישפר/יעצב מחדש את המצגת הקיימת שלכם.
מסמכים ודוחות
PDF, CSV, טקסט — נתונים אמיתיים, מספרים, ציטוטים לשילוב.
פרופיל מותג (יום 6)
צבעים, פונטים, לוגו, טון — עקביות מותגית מיידית.
נתונים ספציפיים
הכנסות, מדדים, מחקר — AI לא ממציא נתונים אמיתיים, אתם צריכים לספק אותם.
חשוב

אם המצגת צריכה נתונים אמיתיים (הכנסות, מדדים, מחקר) — ספקו אותם. מספרים שנוצרו ב-AI הם בדויים. הנתונים שלכם = מצגת מדויקת.

תמונות AI
Hero images ואינפוגרפיקות נוצרים עם Gemini (סקיל יום 1) בצבעי המותג שלכם.
כללי איכות
מילים אסורות, מקסימום 6 נקודות לשקף, פריסות א-סימטריות, מרחב לבן נדיב.

מסגרות סיפוריות

כל מצגת מספרת סיפור

FRAMEWORK 01

"למה זה חשוב" (ברירת מחדל)

Hook → בעיה → פתרון → הוכחה → יישום → פעולה → סיום. הכי טוב לעסקים וחינוך.

FRAMEWORK 02

"המסע"

הקשר → התפתחות → גילוי → צלילה לעומק → השפעה → עתיד → סיום. מושלם להשקות מוצרים.

FRAMEWORK 03

"למד והשראה"

תשומת לב → צורך → סיפוק → ויזואליזציה → פעולה → סיום. מצוין להדרכות ומוטיבציה.

Redesign my current presentation at ~/Documents/old-deck.pptx. Use my brand profile from ~/brand-profiles/mybrand/. Keep the data but improve everything.

עם נתונים

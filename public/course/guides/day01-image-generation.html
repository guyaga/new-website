Day 01

יצירת תמונות עם AI

Gemini 3 Pro Image Preview

בשיעור הזה נלמד ליצור תמונות מקצועיות ישירות מ-Claude Code — פוסטים לסושיאל, אינפוגרפיקות, מוקאפים, ועיצובים מותאמי מותג. בלי פוטושופ, בלי כלי עיצוב, בלי ניסיון בעיצוב. רק מתארים במילים ומקבלים תמונה מוכנה.

מה זה בכלל?

הכירו את Gemini 3 Pro

Gemini 3 Pro הוא מודל AI של גוגל שיודע ליצור ולערוך תמונות ברמה מקצועית. בניגוד לכלים אחרים כמו DALL-E או Midjourney, ל-Gemini יש כמה יתרונות ייחודיים שהופכים אותו למושלם ליצירת תוכן עסקי:

טקסט קריא על תמונות
רוב מודלי AI לא מצליחים לכתוב טקסט ברור על תמונות. Gemini 3 Pro כותב טקסט מדויק, קריא, גם בעברית.
רזולוציה גבוהה — עד 4K
תמונות באיכות הדפסה, לא ממוזערות ומטושטשות. מתאים לפוסטים, מצגות, ואפילו הדפסה.
14 תמונות ייחוס
שלחו את הלוגו שלכם, תמונות מוצר, תמונות אישיות — ה-AI ישתמש בהן ליצירת התמונה.
עריכה בשיחה
צרו תמונה, ואז אמרו "תגדילו את הכותרת" או "תחליפו צבע" — כמו לדבר עם מעצב.
חיפוש גוגל בזמן אמת
צרו ויזואליזציות מבוססות מידע עדכני — מזג אוויר, מניות, אירועים, טרנדים.
חשיבה חכמה
המודל "חושב" לפני שהוא יוצר — מנתח את הבקשה שלכם ומתכנן את התמונה לפני היצירה.

הגדרות

איך להשיג API Key

API Key זה כמו "סיסמה" שמאפשרת ל-Claude Code להתחבר לשירות של גוגל. ככה הוא יודע שזה אתם.

1

כנסו ל-Google AI Studio

לכו ל-aistudio.google.com/apikey והתחברו עם חשבון Google.

2

צרו מפתח API

לחצו על "Create API Key" ובחרו פרויקט (או צרו חדש). המפתח ייווצר תוך שניות.

3

העתיקו את המפתח

לחצו Copy ושמרו אותו. תצטרכו להדביק אותו בשלב הבא.

4

הגדירו את המפתח במחשב

פתחו Terminal (שורת פקודה) והדביקו:

Windows (PowerShell)
$env:GEMINI_API_KEY="your-api-key-here"
macOS / Linux
export GEMINI_API_KEY=your-api-key-here
חשוב מאוד

המפתח הזה הוא אישי — לא לשתף אותו עם אף אחד, ולא להעלות אותו ל-GitHub. תמיד להשתמש בקובץ .env.

התקנה

התקנת הסקיל

הסקיל "nano-banano-pro" הוא מה שמאפשר ל-Claude Code ליצור תמונות. ככה מתקינים:

שלב 1: התקנת ה-SDK

פתחו Terminal בתיקיית הפרויקט שלכם והריצו:

Terminal
npm install @google/genai

שלב 2: התקנת הסקיל

אם הסקיל לא מותקן עדיין, העתיקו אותו לתיקיית הסקילים:

Terminal
cd ~/.claude/skills/ git clone https://github.com/guyaga/10d10s-day01-image-generation nano-banano-pro

טיפים

איך לקבל תוצאות טובות יותר

1

היו ספציפיים

"פוסט לסושיאל בסגנון שוויצרי עם טיפוגרפיה אדומה בולדית על רקע שחור" עדיף על "תעשו לי תמונה יפה".

2

ציינו צבעים לפי קוד

במקום "אדום" כתבו "#E63B2E" — ככה תקבלו בדיוק את הצבע שאתם רוצים.

3

תארו את הפריסה

"לוגו למעלה משמאל, כותרת במרכז, תמונה בצד ימין" — ככה ה-AI מבין איפה לשים דברים.

4

תמיד 2K לפחות

לעולם אל תשתמשו ב-1K. ההבדל באיכות עצום. 2K הוא המינימום לתוצאה מקצועית.

5

שלחו תמונות ייחוס

רוצים שהתמונה תיראה כמו משהו מסוים? שלחו דוגמה. ה-AI יבין את הסגנון.

פתרון בעיות

משהו לא עובד?

?

"API key not found"

ודאו שהגדרתם את GEMINI_API_KEY ב-Terminal או בקובץ .env. אם סגרתם את ה-Terminal, צריך להגדיר שוב.

?

התמונה מטושטשת

שנו את imageSize מ-1K ל-2K. תמיד להשתמש ב-2K לפחות.

?

הטקסט לא קריא

הוסיפו לפרומפט: "text must be perfectly legible and large". ציינו גם את הפונט והגודל הרצוי.

?

הצבעים לא נכונים

ציינו את הצבעים לפי קוד hex מדויק בפרומפט. לדוגמה: "ONLY use #111111 and #E63B2E".

?

חריגה מהמכסה

בדקו את הצריכה שלכם ב-Google AI Studio. בדרך כלל המכסה מתאפסת כל יום.
